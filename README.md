# Asistente radiológico IA-Rx

NO PARA USO MÉDICO. Con el fin de reducir la distancia entre investigadores de IA y profesionales médicos, hemos desarrollado un prototipo muy accesible y gratuito que puede ser utilizado por profesionales sanitarios para comprender la realidad de las herramientas de Deep Learning aplicadas al diagnóstico de radiografías de tórax. El sistema está diseñado como una segunda opinión, donde el usuario puede procesar una imagen para confirmar o apoyar su diagnóstico. La herramienta predice 18 hallazgos radiológicos diferentes basándose en datos de los 7 mayores conjuntos de datos públicos. 
Lo que hace única a esta herramienta es que la versión web se ejecuta completamente de forma local y no envía ningún dato fuera del dispositivo, lo que permite que pueda escalar a millones de usuarios de manera gratuita. La herramienta está disponible como página web, compatible con ordenadores y teléfonos móviles, y con nuestra nueva versión 3 ofrecemos versiones nativas para Windows y Mac

Los objetivos de este sistema son:
# Demostrar funcionamiento y limitaciones — Mostrar cómo funcionan los sistemas de IA y cuáles son sus limitaciones reales.
# Potenciar los datos abiertos — Evidenciar el potencial de los datos abiertos, necesarios para construir un sistema público como este.
# Crear una herramienta educativa — Desarrollar un recurso que ayude a enseñar radiología.
Demostrar un sistema escalable — Mostrar un sistema de distribución de modelos capaz de escalar y ofrecer herramientas médicas gratuitas a nivel global.
